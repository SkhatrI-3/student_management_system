## Project Description
This is a **Student Management System** built with **Laravel**.  
It allows managing **teachers, students, assignments, submissions, and attendance**.  
Main features:
- Add/Edit/Delete teachers and students
- Upload and manage assignments
- Record and track attendance
- View submissions per student and teacher

## Technologies Used
- PHP 8.x
- Laravel 10.x
- MySQL
- Blade Templates

## Installation
1. Clone the repository  
2. Run `composer install`  
3. Configure `.env` file  
4. Run migrations: `php artisan migrate`
